# Backend server for Life-App

### Seting up the server

```bash
# ENVS

- HOST (for database)
- PORT (for db)
- NAME (name of db user)
- PASS (db password)
- DATABASE (db name)
- DEV (is development )
- SYNC (synchronize database schema, not recommended for prod)
- APP_PORT (PORT for server)
- NODE_ENV


```
